Detailed Description

A structure to hold mesh data.

This structure is BlueprintType, meaning it can be used in Blueprint. It contains an ID of type FString, a description of type FString, and a triangle index array of type TArray<int32>. The ID is used to uniquely identify the mesh data. The triangle index array is used to store the indices of the triangles that make up the mesh.
