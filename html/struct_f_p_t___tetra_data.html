Detailed Description

A structure that represents tetrahedron data.

This structure is defined as a BlueprintType, which means it can be used in Blueprints. It contains four double values (A, B, C, D) representing the vertices of a tetrahedron, a string (Tag) for tagging or labeling the tetrahedron, and an index for identification.
